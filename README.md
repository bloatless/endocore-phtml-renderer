# endocore-phtml-renderer

PHtml Renderer component for the Endocore framework.

### Output

```
{{ $foo }}

{{ $foo|raw }}
```

### Conditions

```
{% if ($foo === true) %}
    ...
{% elseif ($bar === true) %}
    ...
{% else %}
    ...
{% endif %}
```

### Includes

```
{{ include('some/view') }}
```
